# Zava Smart Order Feasibility — A2A Multi-Agent Demo

> **Status:** In development. This README is a placeholder and will be finalized after the implementation plan is approved and the demo is built.

## Overview (preliminary)

This repository contains an external customer demo that shows two AI agents collaborating via the open **A2A (Agent-to-Agent)** protocol to solve a real manufacturing problem for the fictional company **Zava** (precision components — industrial pumps and motors).

- **Customer Service Agent** — Microsoft Foundry V2 agent (user-facing).
- **Manufacturing Ops Agent** — LangGraph application on AKS (internal).

A React UI visualizes every A2A hop and every agent action in real time.
